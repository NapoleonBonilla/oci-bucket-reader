A modern, web-based file browser for Oracle Cloud Infrastructure (OCI) Object Storage buckets using Pre-authenticated Request (PAR) URLs.

How to use: 

1. Download the oci-browser.html file
2. Open it in any modern web browser
3. Enter your OCI PAR URL
4. Start browsing your files

Prerequisites:

1. OCI Object Storage Bucket with a Pre-authenticated Request (PAR) URL
2. Modern Web Browser (Chrome, Firefox, Safari, Edge)
3. HTTPS PAR URL
